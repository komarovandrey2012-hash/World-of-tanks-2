# World Of Tanks 2

Игра в жанре «аркада-лабиринт», где вы управляете танком и сражаетесь с противниками на процедурно генерируемой карте. Уничтожайте врагов, собирайте бонусы и следите за уровнем здоровья.

## ОПИСАНИЕ ИГРЫ

Вы — жёлтый танк (или другой цвета, в зависимости от текстур). Ваша цель — уничтожить все вражеские танки на карте. Враги появляются случайным образом, патрулируют территорию и стреляют в вас. Карта состоит из различных блоков: земля, кирпич, бетон, вода. Кирпичные стены можно разрушать выстрелами. По карте разбросаны бонусы: пополнение боезапаса и временное ускорение.

Игра поддерживает динамическую камеру, которая следует за игроком, а также отображение полоски здоровья над каждым танком.

## УПРАВЛЕНИЕ

### Управление танком

| Действие | Клавиша |
|----------|---------|
| Движение вверх | `W` |
| Движение вниз | `S` |
| Движение влево | `A` |
| Движение вправо | `D` |
| Стрельба | `Пробел` |

### Управление камерой

Камера обычно следует за игроком автоматически, но её можно смещать вручную:

| Действие | Клавиша |
|----------|---------|
| Сдвинуть камеру вверх | `↑` (Стрелка вверх) |
| Сдвинуть камеру вниз | `↓` (Стрелка вниз) |
| Сдвинуть камеру влево | `←` (Стрелка влево) |
| Сдвинуть камеру вправо | `→` (Стрелка вправо) |

## ХОД ИГРЫ

1. Запустите файл `main.py`
2. Игровой танк появляется в центре карты
3. Вокруг случайным образом расставлены вражеские танки
4. Уничтожайте врагов выстрелами с помощью `Пробел`
5. Собирайте бонусы:
   - **Коробка с патронами** — пополняет боезапас на 10 единиц
   - **Молния** — временно увеличивает скорость танка
6. Избегайте воды — в ней танк движется медленнее
7. Пробивайте кирпичные стены выстрелами
8. Следите за полоской здоровья над танком — она меняет цвет по мере урона
9. Когда все враги уничтожены, на экране появляется надпись **Win**
10. Если ваш танк уничтожен — игра заканчивается с надписью **Wasted**

## ОСОБЕННОСТИ

- **Процедурная генерация карты** — каждый новый заезд уникален
- **Динамическая камера** — следит за игроком, можно смещать вручную
- **ИИ противников** — враги преследуют игрока и стреляют
- **Разрушаемые стены** — кирпич исчезает от попадания снаряда
- **Бонусы** — пополнение боезапаса и временное ускорение
- **Индикатор здоровья** — отображается над каждым танком
- **Разные типы препятствий** — бетон (неразрушаемый), кирпич (разрушаемый), вода (замедляет)

## ТИПЫ БЛОКОВ НА КАРТЕ

| Обозначение | Тип | Свойства |
|-------------|-----|----------|
| Земля | Обычная трава | Свободный проход |
| Кирпич | Разрушаемая стена | Разрушается от выстрела |
| Бетон | Неразрушаемая стена | Препятствие |
| Вода | Замедляющая поверхность | Танк едет медленнее в 2 раза |
| Бонус с патронами | Пополнение боезапаса | +10 к боезапасу |
| Бонус ускорения | Временное ускорение | Удваивает скорость |
